# Polygon web wallpaper

This project is my experiment to create abstract background for my future site, but this idea looks more situable for desktop wallpaper.

![Preview image](https://github.com/dd4rky/polygon/blob/main/preview.png?raw=true)


The idea of creating this effect was [Delaunay triangulation](https://en.wikipedia.org/wiki/Delaunay_triangulation)
